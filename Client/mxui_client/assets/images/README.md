# Assets Images

Place your application images here.

## Required Images:

- `logo.png` - Application logo (512x512)
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen background
- `placeholder.png` - Placeholder for servers without images

## Notes:

- Use PNG format with transparency
- Optimize images for mobile (keep file sizes small)
- Use @2x and @3x versions for different screen densities if needed
