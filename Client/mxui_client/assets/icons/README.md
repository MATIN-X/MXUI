# App Icons

Place app icons here for different platforms.

## Android Icons:

Generate using: `flutter pub run flutter_launcher_icons`

Required sizes:
- mipmap-mdpi (48x48)
- mipmap-hdpi (72x72)
- mipmap-xhdpi (96x96)
- mipmap-xxhdpi (144x144)
- mipmap-xxxhdpi (192x192)

## iOS Icons:

Required sizes (see ios/Runner/Assets.xcassets/AppIcon.appiconset/)
- 20x20, 29x29, 40x40, 60x60, 76x76, 83.5x83.5, 1024x1024

## Web:

- favicon.png (16x16, 32x32, 192x192, 512x512)
